# AI Influencer System

## Concept
Build many niche AI influencers that post educational short-form content at scale.
Monetize via $20–$30 info products. Funnel traffic using lead magnets embedded in content.

## Stack (TBD — pending research)
- **Video generation**: RunPod + self-hosted models (candidates: InfinityTalk, HeyGen alternatives)
- **Multi-account management**: Multilogin
- **Scheduling/distribution**: TBD (Publer / other)
- **Platforms**: YouTube Shorts, Instagram Reels, TikTok

## Monetization Model
- Dense $20–$30 info product per niche
- Multiple posts/day per account
- Some posts = pure value, some = lead magnets → product

## Folder Structure
- `research/` — niche research, tool comparisons, platform research
- `content/` — scripts, templates, repurposing frameworks
- `infrastructure/` — setup guides, configs, automation
- `products/` — info product drafts and outlines
- `accounts/` — account tracking (gitignored)
