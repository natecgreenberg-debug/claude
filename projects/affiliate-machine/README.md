# Recurring SaaS Affiliate Pipeline

Milestone-based system for building recurring affiliate income through SaaS programs using free traffic methods. Zero upfront cost.

## Milestones

### Milestone 1: First Sale (Target: Month 2-3)
- Apply to all Tier 1 programs (8 programs, instant/easy approval)
- Publish 5+ comparison/review articles on Medium + blog
- Build LinkedIn presence (3-5 posts/week)
- Build Reddit karma (500+, no promotion yet)
- Answer 10+ Quora questions with blog post links

### Milestone 2: $100/mo Recurring (Target: Month 4-6)
- 30+ articles published across platforms
- YouTube faceless channel launched (10+ videos)
- Apply to Tier 2 programs (ClickFunnels, Kajabi, Leadpages)
- n8n automation deployed (content scheduling, analytics)
- 3-4 active referrals generating recurring commissions

### Milestone 3: $1,000/mo Recurring (Target: Month 12-18)
- 50+ articles, SEO traffic compounding
- YouTube channel approaching monetization (1K subs)
- Apply to Tier 3 programs (SEMrush, Kinsta, HubSpot)
- 30-40 active referrals across multiple programs
- Consider multi-account scaling (only if revenue justifies)

## Content Strategy
1. **Comparison articles** ("X vs Y") — highest conversion format
2. **Honest reviews** — builds trust with decision-stage buyers
3. **"Best X for Y" lists** — captures broad search traffic
4. **Tutorials** — provides value, natural tool recommendations
5. **YouTube faceless** — tool reviews, screen recordings, comparisons

## Platform Priority
1. LinkedIn (B2B, highest organic engagement, most AI-permissive)
2. Reddit (authority building, requires karma investment)
3. YouTube (long-term compound, affiliate links from day 1)
4. Medium (SEO via high DA, publication distribution)
5. Quora (long-tail SEO + AI search visibility)
6. TikTok/Instagram (short-form reach, content repurposing)

## Tech Stack ($0 Total)
