# Affiliate Machine

High-ticket/recurring commission affiliate marketing using free traffic methods (SEO, Medium, YouTube, Reddit, Quora, LinkedIn).

## Strategy
- Promote SaaS tools with 25-60% lifetime recurring commissions
- Create comparison/review content with Claude, publish across free platforms
- Compound recurring revenue: each referral earns every month the customer stays subscribed
- Zero upfront cost -- all free platforms and tools

## Target Programs (Easy Acceptance, Lifetime Recurring)

| Program | Commission | Cookie | Plan Price |
|---------|-----------|--------|------------|
| Systeme.io | 60% | 365 days | $27-97/mo |
| GetResponse | 33-60% | 120 days | $19-119/mo |
| AWeber | 30-50% | 365 days | $15-899/mo |
| ConvertKit | 30% | 30 days | $9-2,000+/mo |
| Mangools | 30% | 30 days | $30-90/mo |

## Content Strategy
1. "Best X for Y" comparison articles (highest conversion: 3-8%)
2. "[Tool A] vs [Tool B]" head-to-head reviews
3. "How to [solve problem] using [tool]" tutorials
4. Faceless YouTube screen-recording walkthroughs

## Platforms
- **Hashnode** (free blog with custom domain, strong SEO)
- **Medium** (DA 96, fast ranking, affiliate links allowed with disclosure)
- **YouTube** (faceless tutorials, affiliate links from day 1)
- **Quora** (answers linking to blog as bridge pages)
- **Reddit** (karma first, 90/10 value ratio, link to blog not affiliate URLs)
- **LinkedIn** (B2B SaaS content, carousels, links in comments)

## Tech Stack
- Claude Code: content generation
- n8n: scheduling, repurposing, tracking workflows
- Hashnode: free hosted blog
- Google Sheets: tracking affiliate links, clicks, conversions
- Free analytics: Medium stats, YouTube Studio, Google Search Console
